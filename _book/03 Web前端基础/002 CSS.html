002 CSS

[TOC]

概念

理解：装饰

样式：颜色、字体、距离、定位...

语法

<!DOCTYPE html>
<html lang="en">
    <head>
        <style>
            选择器 {css属性:样式;}
        </style>
    </head>

    <body>
    </body>
</html>

示例一

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>鸣人学院</title>

    <style>
        p{
            color: red;
        }
    </style>
</head>
<body>
你好
<p id="content">欢迎来到</p>
</body>
</html>

示例二

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>鸣人学院</title>

    <style>
        p{
            color: red;
        }

        body{
            background-color: green;
        }
    </style>
</head>
<body>
你好
<p id="content">欢迎来到</p>
</body>
</html>

示例三

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>鸣人学院</title>

    <style>
        *{
            padding: 0;
            margin: auto;
        }

        p{
            color: red;
        }

        body{
            background-color: green;
        }
        .btn{
            background-color: blue;
            color: white;
        }
    </style>
</head>
<body>
你好
<p id="content">欢迎来到鸣人学院</p>

<form action="">
    用户名: <input type="text" name="" id="">
    密码: <input type="password" name="" id="">
    <button class="btn">登录</button>
</form>
</body>
</html>
