## <font color="#0099CC">html结构</font>

```html

<html lang="en">
    <head>
    </head>

    <body>

```

## <font color="#0099CC">第一个html页面</font>

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<h1>我的标题</h1>
    欢迎来到鸣人学院
</body>
</html>
```

## <font color="#0099CC">属性</font>

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<h1>我的标题</h1>
你好
<p id="content">欢迎来到鸣人学院</p>
</body>
</html>
```
